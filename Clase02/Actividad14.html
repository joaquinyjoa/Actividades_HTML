¿Para qué sirve el selector universal * en CSS?

Aplica estilos a todos los elementos HTML de la página, ideal para dar formato base general.

¿Qué es una pseudoclase como :hover?

s una forma de cambiar el estilo según el estado del elemento, como al pasar el mouse.

¿Cuál es la diferencia entre id y class?

id se usa para un solo elemento único, class para varios elementos con el mismo estilo.
